# Bundled font

**OpenDyslexic** (Regular + Bold, woff2) — used by the "Easy-read font" accessibility
option (see `src/a11y.js` / `:root.dyslexia` in `style.css`). The font loads lazily, so
it's only downloaded when a player turns the option on.

OpenDyslexic is a free, open typeface designed to increase readability for readers with
dyslexia — https://opendyslexic.org. Bundled under its open license; attribution above.
